🍽️ Hostel Mess Food Review

A web app where students can view today's mess menu 
and rate each meal in real time.

## Tech Stack
- Python / Django
- SQLite
- Bootstrap 5
- Chart.js

## Setup Instructions
```bash
git clone https://github.com/yourusername/mess-food-review.git
cd mess-food-review
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
